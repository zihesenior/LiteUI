# LiteUI
可以运行在电脑、手机、游戏以及嵌入式设备上的界面框架

开发中，计划支持C#+Xaml以及lua+xaml进行开发，C#+xaml版本已经在android/windows以及Stride3D游戏引擎中跑通

开源计划：计划开源，正在整理中...
